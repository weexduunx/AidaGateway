# Changelog

All notable changes to `laravel-aida-gateway` will be documented in this file.

## 1.0.0 - 2025-11-05

### Added
- Initial release
- Support for Orange Money gateway
- Support for Wave gateway
- Support for Free Money gateway
- Support for E-money gateway
- Unified API for all gateways
- Automatic webhook handling
- Transaction security middleware
- Payment events (PaymentSuccessful, PaymentFailed, PaymentPending)
- Transaction model with Eloquent
- Comprehensive logging
- Facade support (Aida::pay(), Aida::checkStatus(), Aida::refund())
- Full documentation and examples
